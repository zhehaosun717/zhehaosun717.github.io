Game Environment

Atmospheric environment design with focus on lighting and spatial storytelling

Project Overview

This environment design project focuses on creating atmospheric spaces that tell stories through environmental cues. The main goal was to develop a consistent artistic language that conveys narrative and emotion without relying on explicit exposition.

Inspired by traditional Chinese architecture and natural environments, the project explores how cultural elements can be integrated into virtual spaces while maintaining gameplay functionality. Special attention was given to lighting design, which plays a crucial role in guiding the player through the space.

The environment uses a combination of hand-painted and procedural textures to achieve a unique visual style that balances realism with artistic expression. Dynamic elements like particle effects, ambient animations, and responsive lighting enhance the immersive quality of the space.

Project Details

Technologies

  • Unreal Engine 5
  • Nanite Virtualized Geometry
  • Lumen Global Illumination
  • Substance Designer
  • ZBrush for Environmental Assets

Duration

4 months (2023)

Environment Perspectives

Technical Highlights

Dynamic Lighting System

Implemented a time-of-day system that simulates realistic lighting conditions, affecting not only visual appearance but also gameplay mechanics. The environment responds to lighting changes, revealing or hiding elements based on the current lighting state.

Material Design

Created a custom material system that realistically simulates various surface properties like weathering, moisture, and reflection. Materials dynamically respond to environmental factors, showing wear patterns or moisture accumulation in a physically plausible way.
