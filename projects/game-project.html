Game Project

Full game development project with innovative mechanics and storytelling

Project Overview

This game project explores the intersection of traditional Chinese mythology and modern gameplay mechanics, creating a unique narrative experience that bridges cultural elements with contemporary game design principles.

The game features a protagonist who navigates between two worlds - the ancient mythological realm and a modern urban setting. The core gameplay mechanic involves manipulating elements from both worlds to solve puzzles and progress through the story.

Special attention was given to art direction, ensuring that the visual language maintains authenticity while remaining accessible to a global audience. The sound design creates an immersive atmosphere that shifts seamlessly between the two worlds.

Project Details

Technologies

  • Unity Engine
  • C# Programming
  • 3D Modeling (Maya)
  • FMOD Audio Integration
  • Shader Graph

Team Size

5 members (Lead Designer, Programmer, Artist, Animator, Sound Designer)

Duration

10 months (2022-2023)
