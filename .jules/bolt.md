## 2025-02-23 - Conditional Math.sqrt for Continuous Gradients
**Learning:** When optimizing distance calculations with squared thresholds to avoid `Math.sqrt` in performance-critical animation loops (like in `js/fluid-distortion.js`), the actual square root must still be conditionally computed if the original logic relies on it for continuous mathematical gradients or smooth visual falloffs (e.g., lerping or continuous influence). Simply substituting squared distance into the linear formula breaks the visual smoothness.
**Action:** When converting distance checks to use squared values, only skip the `Math.sqrt` entirely if checking a boolean condition (e.g., is inside/outside radius). If the distance is needed for a continuous scale (like `1 - dist / radius`), compute `Math.sqrt` inside the `if (distSq < radiusSq)` block.
